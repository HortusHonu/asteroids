# Asteroids Game

A clone of the classic Asteroids game

## Description

Asteroids "built from scratch" with the help of pygame.

## Getting Started

### Dependencies

* pip
* Python3
* Pygame

### Installing

* python --version

* pip --version

* pip install pygame

### Executing program

* python3 main.py

## Help

* Run from Linux or WSL
